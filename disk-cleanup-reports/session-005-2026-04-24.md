## Structural Fixes to Prevent Fast Regrowth

| Fix | Impact | How |
|-----|--------|-----|
| Disable macOS Aerial screensaver | Stops 4–6G per macOS update | System Settings → Screen Saver → pick non-aerial |
| Quit Docker Desktop when not using grobiz | 8.9G Docker.raw stays but doesn't grow | Cmd+Q Docker Desktop |
| Quit Claude Desktop when not using | Prevents VM re-expansion (was 10G) | Already deleted vm_bundles |
| VS Code update installers | ~800M per update | Monthly: `rm -rf ~/Library/Caches/com.microsoft.VSCode.ShipIt` |
