# Session 26 — Same-Day Regrowth After S25, Gradle Named as New Offender

**Date:** 2026-07-09
**Disk before:** 92% used — 20GB free
**Disk after:** 86% used — 32GB free
**Recovered:** ~12GB

---

## Context

Session 25 ended at 09:13 this morning with 38GB free (83% used). By 22:21 the same day, free space had dropped to 20GB (92% used) — an 18GB regression in roughly 13 hours, with no reboot or emergency in between. This session diagnosed the regrowth before cleaning, since a same-day 18GB drop was unusual enough to warrant checking for something other than the usual build-cache regrowth (e.g. Docker.raw ballooning, a runaway process, snapshots).

Diagnosis ruled out: local Time Machine snapshots (none present), Docker.raw growth (still 8.0GB actual/sparse, `ls -lh`'s 228GB is logical/apparent size only — not a red flag), and `klo`/`awinta` accounts (both stable). The actual cause was mundane: a mobile build/test run this afternoon (~15:00–18:07) regenerated every cache S25 had just wiped, plus grew `~/.gradle` — never tracked by name in prior sessions — to 4.4GB.

## Actions Taken

| # | Item | Command | Recovered |
|---|------|---------|-----------|
| 1 | Simulator shutdown (blocker) | `xcrun simctl shutdown all` | — (unblocked step 3) |
| 2 | Xcode DerivedData | `rm -rf ~/Library/Developer/Xcode/DerivedData/*` | 2.7GB → 0B |
| 3 | CoreSimulator device content | `xcrun simctl erase all` | 3.3GB → 192MB |
| 4 | Gradle caches | `rm -rf ~/.gradle/caches/*` | 4.4GB → 0B |
| 5 | CocoaPods + Playwright caches | `rm -rf ~/Library/Caches/{CocoaPods,ms-playwright,ms-playwright-mcp}` | 941MB + 568MB |
| 6 | npm cache | `rm -rf ~/.npm/_npx ~/.npm/_cacache` | ~400MB → 52KB |
| 7 | dotslash cache | `rm -rf ~/Library/Caches/dotslash` | Partial — `Permission denied` on signed app bundle internals (same as S25) |
| 8 | dotslash cache (retry) | `sudo rm -rf ~/Library/Caches/dotslash` | Remainder → 0B |
| 9 | Docker unused images | `docker image prune -a -f` | 1.642GB (running grobiz stack, 7 containers, untouched) |

All steps pre-approved as a single batch after the recommendation was presented.
