# Session 22 — Gradle/DerivedData/Simulator/Docker Sweep

**Date:** 2026-07-08
**Disk before:** 93.8% used — 15.2GB free
**Disk after:** 86.5% used — 33.2GB free

---

## Context

User asked for a general disk cleanup. Disk had degraded fast: 31GB free on 2026-07-05 (S21) down to 15.2GB free three days later — worse than the "before" state of most prior sessions. No single new offender explained it; it was broad regrowth across several known repeat offenders plus one previously untracked line item (Android SDK, judged not to be bloat — see below).

## Actions Taken

| # | Item | Command | Recovered |
|---|------|---------|-----------|
| 1 | pnpm store prune | `pnpm store prune` | ~1.8GB (841 orphaned packages, 26,518 files) |
| 2 | Gradle caches | `rm -rf ~/.gradle/caches` | 5.4GB (5.6GB → 215MB) |
| 3 | Xcode DerivedData | `rm -rf ~/Library/Developer/Xcode/DerivedData/*` | 2.8GB (→ 0B) |
| 4 | npm cache | `npm cache clean --force` | ~1.6GB |
| 5 | Chrome Service Worker CacheStorage/ScriptCache (6 profiles: Default, Profile 1/2/3/6/8) | `rm -rf` targeted subfolders | ~2.1GB (Chrome was running — files it still had open won't fully free until next restart) |
| 6 | Docker prune (images + stopped containers + build cache, **no volumes**) | `docker system prune -a -f` | 2.37GB |
| 7 | CoreSimulator device content (11 devices) | `xcrun simctl erase all` | 2.3GB (2.5GB → 192MB) — device definitions kept |

Docker: grobiz's compose stack (`grobiz-nginx-1`, `grobiz-scheduler-1`, `grobiz-worker-1`, `grobiz-socketio-1`, `grobiz-erpnext-1`, `grobiz-redis-1`, `grobiz-mariadb-1`) was actively running throughout (started ~15 min prior) — none of the pruned images (`pgvector:pg16/pg17`, `pasarbesar-roadmap:verify`, `redis:8-alpine`, `postgres:17-alpine`, `minio/minio`, `testcontainers/ryuk`) were grobiz-related, and prune never touches images backing running containers regardless. Deliberately skipped `--volumes` — named volumes like `infra_eventopia_pg` may hold real local dev-database data that's expensive to lose; left all volumes untouched.
