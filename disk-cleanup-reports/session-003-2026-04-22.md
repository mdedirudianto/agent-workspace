## Notes

### iOS Simulator regrowth (recurring)
`xcrun simctl delete unavailable` ran but found 0 unavailable devices — all 11 simulators are iOS 26.3 active devices. The 4.4G in `EDCBA20F` (iPhone 17 Pro) is from app data installs. Use `xcrun simctl erase EDCBA20F-CBB4-403C-91D9-9516B74C8ED5` to clear just its data without removing the device, or leave it for grobiz mobile testing.

### Docker (1.5G) — pending
Docker Desktop was not running. Run when active:
```bash
docker system prune -a -f
```
This removes all unused images, stopped containers, and networks (not volumes). Expect 0.5–1.5G recovery.

### npm cache ownership issue
`~/.npm` had root-owned files from a prior `sudo npm` run. Fixed permanently with:
```bash
sudo chown -R $(id -u):$(id -g) ~/.npm
```
