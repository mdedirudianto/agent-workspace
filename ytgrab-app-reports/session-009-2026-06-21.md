### Build gotcha — `api-client` is a dep of **web**, not api
`pnpm --filter "@ytgrab/api..."` builds api + *its* deps (contracts/db/auth) but **not** `api-client`, which only `web` consumes. So web compiled against a stale `api-client/dist` missing the new `drainMergeQueue` method → `tsc` error in `apps/web/src/app/admin/actions.ts:46`. Fix: build web with the **`...` dependency suffix** (`@ytgrab/web...`) so `api-client` rebuilds first. **Lesson for next deploy:** when shared packages change, build each app with its `...` dep closure (or `pnpm -w build`) — don't filter to a single app.
