## Findings / decisions

- **Premise corrected:** hoteru was undeployed → this was a full first deploy, not a proxy-only change.
- **Repo has 4 live apps**, incl. a dedicated `marketing` app (projects-report listed only 3 — stale).
- **No per-hotel multi-tenancy in code yet** (no host→hotel routing; build-time API URLs). Per user
  decision, `*.hoteru.co.id` is **reserved** (wildcard cert + holding page); live tenancy is a
  follow-up app feature. See README "Per-hotel tenancy" + Open follow-ups.
- **DNS is orange-cloud**, so CF SSL/TLS mode must be **Full (strict)** — set manually by the user
  (the provided API token is DNS-scoped only). E2E 200s confirm CF↔origin works (no redirect loop).

## Manual interventions (user)
