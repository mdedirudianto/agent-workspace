# Session 2 — Rename `teach` portal → `teacher` (naming consistency)

**Date:** 2026-06-16
**App:** `seenow` (github.com/biji-dev/seenow) — branch `refactor/teach-portal-to-teacher`, **PR [#1](https://github.com/biji-dev/seenow/pull/1)**
**Host:** `dev` — `deploy@154.26.130.96` (branch deployed to dev ahead of merge)
**Scope:** code + deploy. Follows [session-001](session-001-2026-06-16.md).

## Why

The portal set was inconsistently named: **`learner`** (role-noun) vs **`teach`** (verb).
The codebase's RBAC vocabulary is role-nouns throughout — `learner`, `teacher`, `author`,
`b2b_admin`, `platform_admin` — and the `admin` portal is also a role-noun. The `teach` portal
was the only verb. Standardized on **role-nouns** → `learner` + `teacher` + `b2b` + `admin`.
(`b2b` stays — it's an audience term fronting `b2b_admin`, no clean person-noun.)

## What changed (repo)

- `apps/web-teach` → **`apps/web-teacher`** (`git mv`); package **`@seenow/web-teacher`** + description.
- **Portal-key wire value `"teach"` → `"teacher"`** — this is the value in `GET /v1/me`'s `portals`
  field (per `portals.ts`: *"Keys match the `portals` field of `GET /v1/me`"*). Renamed in lockstep:
  `permissions.ts` (`Portal` type, `PORTAL_ROLES`, `portalsForIdentity`), `routes/me.ts` `PORTALS`
  set, the web-public portal picker (`portal/page.tsx`, `portals.ts`), and the i18n dict label keys
  (`dict.ts` — keys renamed, the verb labels "Teach"/"Mengajar" kept). **Computed from role
  assignments — no DB/migration change.** API + client ship together, so the wire stays coherent.
- Env keys `NEXT_PUBLIC_TEACH_URL`/`VITE_TEACH_URL`/`E2E_TEACH_URL` → `*_TEACHER_URL`;
  web-b2b `TEACH_PORTAL_URL` → `TEACHER_PORTAL_URL`.
- Teacher SPA self-identifies as `GET /v1/me?portal=teacher`.
- Subdomain references `teach.seenow.id` → `teacher.seenow.id` (docs + nginx template).

**Intentionally NOT changed:** the `/v1/teach/*` REST route namespace (a separate axis from the
portal name — renaming it would cascade through the generated api-client/openapi; out of scope for
portal naming). Marketing verb copy ("Assess before you teach", "then teach.") kept.

103 files changed. `bun.lock` regenerated for the package rename and folded into the commit.
