## What was provisioned

- **DB (`db` 10.0.0.1, PG16.9):** db `seenow` + role `seenow` (`LOGIN CREATEROLE BYPASSRLS`, **not**
  superuser — generated password, `.env` only); `vector 0.8.2` pre-created as postgres. Migrations
  created the non-superuser `seenow_app` role (RLS binds, `APP_DATABASE_URL`). **Redis db 7** (db0/2/3/4/6
  in use). `pg_hba` already allows `10.0.0.0/22` (md5) so app reaches it — no hba change. 57 tables,
  RLS deploy gate **27/27**. **No seed.**
- **app (`devops`, `~/seenow`):** `git clone main`; `bun install` (Bun 1.3.14 at `~/.bun/bin/bun`); prod
  `.env` (freshly generated crypto — **not** reused from dev); `bun run build` (7/7).
- **PM2 (`devops`, `~/seenow.config.cjs`):** `seenow-api` (Bun `:4900`), `seenow-workers` (Bun, Redis
  db7), `seenow-web-public` (`next start -H 10.0.0.5 -p 4901`). `NODE_ENV=production`, `pm2 save`,
  **0 restarts**. web-public bound to private `10.0.0.5`; the Bun API binds `0.0.0.0:4900` but is
  **UFW-protected** (public iface exposes only 22/80/443; `10.0.0.0/24` allowed) — same posture as
  eventopia-api (follow-up to bind `10.0.0.5`).
- **Static:** the 4 Vite SPA `dist/`s published from app → proxy `/var/www/seenow/web-{learner,teacher,b2b,admin}`.
- **TLS:** LE wildcard `seenow.ac` + `*.seenow.ac` via **DNS-01**. The `seenow.ac` CF token (same one used
  for dev) stored at `/etc/letsencrypt/cloudflare-seenow.ini` on proxy (0600). Auto-renew scheduled.
- **proxy nginx** (`sites-available/seenow.conf`): upstreams → `10.0.0.5:4900/4901`; blocks
  apex/www/api/learner/teacher/b2b/admin + `*.seenow.ac`; per-vhost `$seenow_conn` upgrade map; api block
  carries WS + tutor-SSE (buffering off); `X-Forwarded-Proto https`.
- **Cloudflare DNS:** the prod records **already existed** — apex/www/api/learner/teacher/b2b/admin **and**
  `*.seenow.ac`, all **proxied (orange)** → `46.250.234.153`. The dev records (`dev.seenow.ac`,
  `*.dev.seenow.ac` → `154.26.130.96`, grey) are untouched. (The DNS-scoped token can't read the zone SSL
  mode, but Full-strict is confirmed working empirically — see smoke.)
