# meetgrab App Reports

App-scoped reports for the **meetgrab** product (`meetgrab.us`) — a Chrome extension that turns Google Meet captions into AI meeting summaries, backed by a stateless LLM-proxy API and an Astro marketing site.

**Repo:** `github.com/biji-dev/meetgrab` (pnpm + turbo monorepo). Apps: `api` (Hono/Node), `web` (Astro static), `admin` (Vite SPA — **not deployed yet**), `extension` (Chrome Web Store — not server-deployed). Packages: `shared-types`, `prompts`, `ui`.

**Server footprint:**
- `app` (`154.26.129.104` / `10.0.0.5`) — `meetgrab-api` under **PM2** (Hono on Node), **private-bound `10.0.0.5:8930`**, `node --env-file`. Source at `~/meetgrab` (devops). **No Docker** (repo ships a Dockerfile/compose for a DO droplet; we run native PM2 per app-server convention).
- `proxy` (`46.250.234.153`) — public nginx; terminates TLS for `meetgrab.us`, `www`, `api.meetgrab.us`. Serves the Astro static build from `/var/www/meetgrab.us`; reverse-proxies `api.` → `10.0.0.5:8930`. CF orange-cloud → Full(strict). LE cert via DNS-01 (`/etc/letsencrypt/cloudflare-meetgrab.ini`).
- `monitor` (`10.0.0.3`) — GlitchTip (`errors.biji.uk`, API project **14**), OpenPanel (`analytics.biji.uk`, shared **MeetGrab Web** project), OpenObserve (logs via `app` fluent-bit glob).
- `db` (`10.0.0.1`) — **none.** The API is stateless (in-memory per-IP rate limiting). No Postgres/Redis.

**Architecture note (subdomain, not `/api`):** the API is a separate subdomain `api.meetgrab.us`. The extension ships pointing at it; CORS in `apps/api/src/app.ts` whitelists `chrome-extension://<32-char>` + `https://meetgrab.us` only. The marketing site does **not** call the API.

**Current deployed SHA:** **`91309a1`** (main, 2026-06-24) — api **v0.5.0** (**async summary queue**: `POST /v1/summaries`→`202 {jobId}` + `GET /v1/summaries/:jobId` poll; adaptive Haiku→Sonnet escalation), web Concept v3 (**English-default at root**, Bahasa at `/id/*`, unchanged since `5d14025`). API redeployed session-005 (51-commit catch-up across the sync→async rewrite); proxy nginx + web untouched.
