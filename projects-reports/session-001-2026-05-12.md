| Dev/Staging | `deploy@dev` | TBD | `deploy` | TBD |

**Standard deploy flow (no CI):**
```bash
# 1. Push/rsync code to server
rsync -av --exclude node_modules --exclude .git . devops@app:/home/devops/<dir>/

# 2. SSH in and rebuild
ssh devops@app "cd /home/devops/<dir> && npm install && npm run build && pm2 restart <name>"
```

Projects that have a git remote can use `git pull` instead of rsync.
