## Fluent Bit Config Summary

**Inputs:**

| Tag | Source | Parser |
|-----|--------|--------|
| `nginx.access` | `/var/log/nginx/access.log` | `nginx` (built-in) |
| `nginx.error` | `/var/log/nginx/error.log` | none (raw text) |

**Filter:** `record_modifier` adds `server=proxy`, `host=proxy.biji.uk` to every record.

**Output:** `http` → `http://10.0.0.3:5080/api/default/nginx/_json` — Basic auth `admin@biji.uk`, gzip, flush every 5s.

**Record shape (access log):**
```json
{
  "_timestamp": 1779638546176106,
  "server": "proxy",
  "host": "proxy.biji.uk",
  "remote": "104.23.175.170",
  "method": "GET",
  "path": "/",
  "code": "200",
  "size": "138",
  "referer": "-",
  "agent": "Uptime-Kuma/2.1.3"
}
```

- Fields `server` and `host` are pre-tagged so future servers shipping to the same stream can be filtered by source
