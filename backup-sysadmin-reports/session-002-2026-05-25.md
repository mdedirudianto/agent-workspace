## What Gets Shipped

### `backup.jobs` → `/var/log/backup.log`

All three cron scripts append to this single file:

| Script | Schedule | Key log lines |
|--------|----------|---------------|
| `backup.sh` | Daily 18:00 UTC | `[backup] Run started`, `[wagrab] Starting dump`, `[tier1] Done`, `[erp-bench] Done`, `[backup] All jobs completed successfully` |
| `rotate.sh` | Sunday 20:00 UTC | `[rotate] wagrab: removed files older than Nd` |
| `verify.sh` | Tuesday 21:00 UTC | Verification results for each dump |

The Lua filter extracts the `job` field from the `[xxx]` prefix. Possible values: `backup`, `wagrab`, `tier1`, `tier2`, `tier3`, `quranchat`, `erp-bench`, `rotate`, `verify`.

### `backup.syslog` → `/var/log/syslog`

System-level events: fail2ban bans, UFW BLOCK events, cron triggers, sshd auth, kernel messages. Verified in OO: UFW BLOCK entries visible immediately (port 23 probe from `102.218.85.149` blocked within minutes of Fluent Bit start).

## Fluent Bit Config

**`/etc/fluent-bit/fluent-bit.conf`:**

```ini
[SERVICE]
    flush        5
    daemon       Off
    log_level    info
    parsers_file parsers.conf
    plugins_file plugins.conf

[INPUT]
    name             tail
    tag              backup.jobs
    path             /var/log/backup.log
    db               /var/lib/fluent-bit/backup-jobs.db
    mem_buf_limit    5MB
    skip_long_lines  On
    refresh_interval 30

[INPUT]
    name             tail
    tag              backup.syslog
    path             /var/log/syslog
    db               /var/lib/fluent-bit/backup-syslog.db
    mem_buf_limit    5MB
    skip_long_lines  On
    refresh_interval 30

[FILTER]
    name   lua
    match  backup.jobs
    script /etc/fluent-bit/backup-extract.lua
    call   extract_job

[FILTER]
    name   record_modifier
    match  *
    record server backup
    record host   backup

[OUTPUT]
    name        http
    match       *
    host        10.0.0.3
    port        5080
    uri         /api/default/backup/_json
    format      json
    http_user   admin@biji.uk
    http_passwd <redacted>
    tls         off
    compress    gzip
```
