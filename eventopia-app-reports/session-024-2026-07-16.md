## Part 2 — New infra: Cloudflare Browser Rendering Worker (AI page-gen screenshots)

The AI page-gen feature's URL-source quality boost (rendering a pasted website to a screenshot so the model sees its real layout, not just extracted text) needs headless Chromium — which the shared prod VPS can't host. User chose **Cloudflare Browser Rendering** (the doc's own recommended path) over the quick-start `thum.io` keyless option or a paid third-party API.

- **New standalone Worker**, not part of the pnpm/turbo monorepo: `infra/cf-screenshot-worker/` (outside `pnpm-workspace.yaml`'s globs on purpose). `src/index.js` — a ~40-line fetch handler: auth-gated (`x-screenshot-key` header vs. a `SCREENSHOT_AUTH_KEY` secret), launches Chromium via the `MYBROWSER` browser binding, viewport screenshot (1280×800 JPEG, not full-page), always closes the browser session in `finally`.
- **Deployed** to `https://eventopia-page-screenshot.biji.workers.dev` under the **BIJI** Cloudflare account, using eventopia's existing `CF_API_TOKEN` — the user added **Workers Scripts:Edit** to that token rather than provisioning a new one.
- **Plan/cost:** Workers **Free** tier's Browser Rendering allotment (10 browser-min/day, 3 concurrent) — confirmed sufficient, no new cost. Upgrade path (Workers Paid) documented if AI page-gen volume ever needs more.
- **Verified live end-to-end**: a request without the auth header → `401`; a request with it, against `https://example.com` → `200`, real 1280×800 JPEG (18KB), viewed and confirmed it rendered the actual page.
- **Not yet committed to git** — left for the user to decide (commit now vs. fold into a later commit).
- Full redeploy/secret-rotation runbook written to `infra/cf-screenshot-worker/README.md`.

1. `resolveTxt('_eventopia-verify.custom-test.biji.uk')` → matched the minted token → **VERIFIED**.
2. `POST /zones/{zone}/custom_hostnames` (real Cloudflare API, `custom_origin_server=custom.eventopia.my`, no SNI pin) → succeeded, **PROVISIONING**.
3. Polled the hostname status: `pending`/`pending_validation` → `active`/`active` after ~36s → **LIVE**.
4. `curl -I https://custom-test.biji.uk` → **`HTTP/2 200`** (not `526`), served by Cloudflare, correctly rewritten to the org's tenant (`x-middleware-rewrite: /_t/malang-creative-fusion`).
