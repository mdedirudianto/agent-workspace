## What was done (both clones, identical flow)

1. Snapshot HEAD; back up `.env` + `ecosystem.config.cjs` (untracked) to `~/eventopia-{dev,prod}-*.bak-20260620`.
2. `git stash` local edits → `git merge --ff-only origin/main` → `1d7cc57`.
3. `pnpm install` (clean; lockfile unchanged — only `apps/api/package.json` moved).
4. `NODE_ENV=production pnpm turbo run build --filter=@eventopia/web-public --filter=@eventopia/console`
   — **both built clean, no `ignoreBuildErrors` needed.**
5. Copy `.next/static` into each standalone bundle (web-public 43 files, console 83) — Next standalone
   omits static.
6. `pm2 restart` the eventopia procs (`--update-env`). **dev:** api/workers/wa-stub/web-public/console.
   **prod:** api/workers/web-public/console (no wa-stub in prod). All online, single restart, no crash loop.

`checkin` was **not** rebuilt — zero changes in the delta (no static re-rsync to proxy needed). `api`
and `workers` run Bun source directly (no build step) but were restarted to pick up the 29 changed api files.
